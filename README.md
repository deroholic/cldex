# cldex
DERO Command Line DEX

## Requires go v1.18

## To build / run:

### 1) Clone repository:
	$ git clone https://github.com/deroholic/cldex.git

### 2) Enter directory:
	$ cd cldex

### 3) Build and run:
	$ go get cldex
	$ go build
	$ ./cldex

## Usage:
```
$ ./cldex --help
cldex [--help] [--wallet=<wallet_file> | <private_key>] [--password=<wallet_password>] [--daemon-address=<127.0.0.1:10102>]
```
